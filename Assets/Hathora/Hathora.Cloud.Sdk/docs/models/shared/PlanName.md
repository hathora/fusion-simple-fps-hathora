# PlanName

A plan defines how much CPU and memory is required to run an instance of your game server.

`tiny`: shared core, 1gb memory

`small`: 1 core, 2gb memory

`medium`: 2 core, 4gb memory

`large`: 4 core, 8gb memory


## Values

| Name     | Value    |
| -------- | -------- |
| `Tiny`   | tiny     |
| `Small`  | small    |
| `Medium` | medium   |
| `Large`  | large    |
